Geometria Plana e Espacial
⚡ Cai todo ano — Alta prioridade
ESSENCIAL
// Tópicos com exercícios

Figuras planas são formas 2D: quadrado, retângulo, triângulo, círculo, trapézio. Você precisa memorizar as fórmulas de área e perímetro de cada uma. O ENEM adora colocar essas fórmulas em contexto de problemas reais (área de terreno, pintura de parede, cerca de jardim).

// Exercícios
Uma sala tem 6m × 4m. Quantos metros quadrados de piso serão necessários?
Uma pizza tem diâmetro de 40cm. Qual a área da pizza? (use π=3,14)
Calcule a área de um triângulo com base 8cm e altura 5cm.
// VÍDEO
▶ Geometria Plana para o ENEM (Professor Ferretto)

Trigonometria relaciona os ângulos de um triângulo retângulo com seus lados. O ENEM cobra muito em contextos práticos: altura de prédio, distância entre pontos, inclinação de rampas. Memorize: seno = oposto/hipotenusa, cosseno = adjacente/hipotenusa, tangente = oposto/adjacente. Macete: SOH-CAH-TOA.

// Exercícios
Um poste de 10m projeta sombra de 10m no chão. Qual é o ângulo de elevação do sol?
Uma rampa de acessibilidade tem 3m de comprimento e faz 30° com o chão. Qual a altura?
Calcule os ângulos e lados de um triângulo 30-60-90 com hipotenusa = 10.
// VÍDEO
▶ Trigonometria para o ENEM do zero (Me Salva!)

Geometria espacial trabalha com sólidos 3D. O ENEM adora problemas de caixa d'água, embalagem, caixas. Fórmulas essenciais: cubo = a³, paralelepípedo = c×l×a, cilindro = π×r²×h, cone = (π×r²×h)/3, esfera = (4/3)×π×r³.

// Exercícios
Uma caixa d'água cilíndrica tem raio 1m e altura 2m. Quantos litros cabe? (1m³ = 1000L)
Uma bola de futebol tem raio 11cm. Qual seu volume? (π=3,14)
Uma caixa retangular tem 50cm × 30cm × 20cm. Quantos cubos de 10cm cabem dentro?
// VÍDEO
▶ Geometria Espacial ENEM (Descomplica)

Funções e Gráficos
⚡ Cai todo ano — Alta prioridade
ESSENCIAL
// Tópicos com exercícios

f(x) = ax + b. É uma reta no plano cartesiano. 'a' é o coeficiente angular (inclinação) e 'b' é o coeficiente linear (onde corta o eixo Y). Muito cobrado em problemas de custo, lucro, velocidade constante e variações lineares.

// Exercícios
Um táxi cobra R$4 de bandeirada + R$2 por km. Escreva a função e calcule o preço de 15km.
Encontre os zeros (raízes) de f(x) = 3x - 9. Esboce o gráfico.
Duas funções se cruzam: f(x) = 2x + 1 e g(x) = -x + 7. Qual o ponto de interseção?
// VÍDEO
▶ Função do 1º Grau para o ENEM (Me Salva!)

f(x) = ax² + bx + c. Forma uma parábola. Para achar as raízes: fórmula de Bhaskara: x = (-b ± √Δ) / 2a, onde Δ = b² - 4ac. O vértice é o ponto máximo ou mínimo da parábola: xv = -b/2a. Muito cobrado em problemas de arremesso, lucro máximo, dimensão otimizada.

// Exercícios
Resolva x² - 5x + 6 = 0 pela Bhaskara. Quantas raízes reais existem?
Uma bola é lançada com altura h(t) = -5t² + 20t + 2. Qual a altura máxima e quando ocorre?
Uma empresa tem lucro L(x) = -x² + 100x - 1500, onde x é a quantidade vendida. Qual o lucro máximo?
// VÍDEO
▶ Função do 2º Grau completa (Ferretto)

PA: diferença constante entre os termos. an = a1 + (n-1)d. Soma: Sn = n(a1+an)/2. PG: razão constante entre os termos. an = a1 × q^(n-1). Soma: Sn = a1(q^n - 1)/(q-1). O ENEM cobra em contextos de juros simples (PA), juros compostos (PG) e crescimento populacional.

// Exercícios
PA com a1=3, d=4. Qual o 10º termo? Qual a soma dos 10 primeiros?
PG com a1=2, q=3. Qual o 5º termo? Qual a soma dos 5 primeiros?
Uma bactéria dobra a cada hora. Começando com 100 bactérias, quantas haverá em 8 horas?
// VÍDEO
▶ PA e PG para o ENEM (Me Salva!)

Probabilidade, Estatística e Combinatória
⚡ Muito cobrado — Média/Alta prioridade
IMPORTANTE
// Tópicos com exercícios

Probabilidade = casos favoráveis / casos possíveis. Sempre um número entre 0 e 1 (ou 0% a 100%). Para eventos independentes: P(A e B) = P(A) × P(B). Para eventos mutuamente exclusivos: P(A ou B) = P(A) + P(B). O ENEM adora contextos com dados, cartas, urnas e sorteios.

// Exercícios
Uma urna tem 3 bolas vermelhas e 7 azuis. Qual a probabilidade de tirar uma vermelha?
Ao lançar um dado e uma moeda, qual a probabilidade de sair 6 e cara?
Um casal quer ter 3 filhos. Qual a probabilidade de todos serem meninas?
// VÍDEO
▶ Probabilidade para o ENEM do zero (Ferretto)

Média: soma de todos os valores ÷ quantidade. Mediana: valor do meio quando ordenado. Moda: valor que aparece mais vezes. O ENEM sempre apresenta gráficos (barras, setores, linha) e pede para interpretar os dados ou calcular essas medidas.

// Exercícios
Notas: 5, 7, 8, 6, 9, 7, 10. Calcule média, mediana e moda.
Um gráfico mostra vendas mensais. Interprete e calcule a média anual.
A média de 5 números é 8. Quatro deles são: 6, 9, 7, 10. Qual é o quinto?
// VÍDEO
▶ Estatística para o ENEM (Me Salva!)

Porcentagem é uma das habilidades mais cobradas no ENEM e na vida real. Juros simples: J = C × i × t. Juros compostos: M = C × (1+i)^t. Desconto, acréscimo, aumento percentual, variação percentual — todos caem toda prova.

// Exercícios
Um produto custa R$200 e teve desconto de 15%. Qual o novo preço?
R$1000 aplicados a juros compostos de 5% ao mês por 3 meses. Qual o montante?
Um salário aumentou de R$2000 para R$2300. Qual foi o aumento percentual?
// VÍDEO
▶ Porcentagem e Juros ENEM (Ferretto)

Interpretação de Texto
⚡ A habilidade mais cobrada do ENEM inteiro
PRIORIDADE #1
// Tópicos com exercícios

A tese é a ideia principal que o autor defende. No ENEM, as questões sempre pedem "de acordo com o texto" ou "o autor defende que...". A resposta ESTÁ no texto — você não precisa saber o assunto de antemão. Leia com atenção ao que está escrito, não ao que você acha.

// Exercícios

// VÍDEO
▶ Como interpretar texto no ENEM (Prof. Noslen)

As mais cobradas: Metáfora (comparação sem "como": "ele é uma rocha"), Metonímia (parte pelo todo: "li Machado de Assis"), Hipérbole (exagero: "estou morrendo de fome"), Ironia (diz o contrário do que pensa), Personificação (atribui vida a algo inanimado), Eufemismo (suaviza algo negativo).

// Exercícios

Redação — A Nota que Muda Tudo

A redação vale 1000 pontos — o mesmo que todas as outras áreas juntas. Invista muito aqui.

// AS 5 COMPETÊNCIAS — O QUE A BANCA AVALIA
COMPETÊNCIA 1
Domínio da norma culta
Ortografia, concordância, pontuação, coesão. Evite erros gramaticais. Leia bastante para internalizar a norma culta.
COMPETÊNCIA 2
Compreender a proposta
Entender o tema e não fugir dele. Muita gente zera por fuga de tema. Leia a proposta com calma antes de começar.
COMPETÊNCIA 3
Seleção de argumentos
Usar dados, fatos, citações e exemplos concretos para sustentar a tese. Evite argumentos vagos como "desde os primórdios...".
COMPETÊNCIA 4
Coesão e coerência
Conectar as ideias com conectivos (portanto, entretanto, ademais, outrossim). Cada parágrafo deve ter ligação com o anterior.
COMPETÊNCIA 5
Proposta de intervenção
A conclusão deve propor UMA solução concreta com: agente, ação, modo/meio, efeito e detalhamento. É um dos critérios mais fáceis de acertar com treino.
// ESTRUTURA DOS 4 PARÁGRAFOS
§1 — INTRODUÇÃO (~7 linhas)
Contextualize o tema com um dado, citação ou afirmação. Apresente sua TESE (o que você vai defender) no final do parágrafo. Nunca copie os textos motivadores.
§2 — 1º ARGUMENTO (~7 linhas)
Seu primeiro argumento. Use dados, pesquisas, citações de filósofos ou especialistas. Explique a causa ou o contexto histórico do problema.
§3 — 2º ARGUMENTO (~7 linhas)
Segundo argumento, diferente do primeiro. Pode explorar as consequências do problema ou uma perspectiva diferente. Use exemplos concretos.
§4 — CONCLUSÃO (~5 linhas)
Retome a tese. Apresente UMA proposta de intervenção completa: Quem faz? O quê? Como? Com que efeito? Ex: "O governo federal deve implementar X por meio de Y, a fim de Z."

Química
Muito cobrada — Alta prioridade
ESSENCIAL
// Tópicos com exercícios

O mol é o "contador" da química: 1 mol = 6,02 × 10²³ partículas (número de Avogadro). Massa molar = soma das massas atômicas dos elementos na fórmula. Estequiometria usa proporções da equação balanceada para calcular quantidades de reagentes e produtos. É o tema mais cobrado em química no ENEM.

// Exercícios

Quantos mols há em 44g de CO₂? Quantas moléculas? (H=1, C=12, O=16)
// VÍDEO
▶ Mol e Estequiometria para o ENEM (Me Salva!)

Termoquímica: reações exotérmicas (liberam calor, ΔH negativo) e endotérmicas (absorvem calor, ΔH positivo). Cinética: fatores que influenciam a velocidade das reações (temperatura, concentração, catalisador, superfície de contato). Equilíbrio: princípio de Le Chatelier — perturbações no sistema deslocam o equilíbrio.

// Exercícios

O que acontece com o equilíbrio de N₂ + 3H₂ ⇌ 2NH₃ se aumentarmos a pressão?
Classifique como exotérmica ou endotérmica: combustão, fotossíntese, queima de vela.
// VÍDEO

// 08 — ESTRATÉGIAS DE PROVA

Dicas Para o Dia da Prova

Saber o conteúdo é 70% do trabalho. Os outros 30% são estratégia na hora da prova.

// DICA #1

Comece pelo que você sabe.

Não fique 30 minutos numa questão difícil. Marque e pule. Responda todas as fáceis primeiro — garante pontos seguros.

// DICA #2

Leia o enunciado até o fim.

60% dos erros no ENEM são por não ler o enunciado completo. Leia devagar. A resposta está no texto — procure antes de usar o que sabe.

// DICA #3

Elimine alternativas absurdas.

Numa questão de 5 alternativas, elimine as 2 claramente erradas. Você passa de 20% para 33% de chance. Ajuda muito nas dúvidas.

// DICA #4

Redação: não deixe em branco.

Redação em branco = zero e risco de eliminação do SISU. Mesmo que ruim, escreva os 4 parágrafos. Qualquer nota é melhor que zero.

// DICA #5

Gerencie o tempo.

Dia 1 (Humanas + Redação): 5h30. Dia 2 (Naturezas + Matemática): 5h30. Média de 3 min por questão. Reserve 1h para redação.

// DICA #6

Faça simulados cronometrados.

Treinar sem tempo é diferente de treinar com tempo. Faça pelo menos 1 simulado completo por mês com cronômetro — como se fosse dia de prova.

// DICA #7

Analise os erros.

Não adianta fazer simulado e olhar só o gabarito. Para cada erro: por que errei? Era conteúdo ou interpretação? Anote e revise esse tópico.

// DICA #8

Cuide do físico.

Sono regular, alimentação e exercício físico têm impacto direto na memória e concentração. Isso não é detalhe — é parte do estudo.
